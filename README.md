# RepulsorPong
A simple pong game created in Unity2D. Play the latest release [here](https://zioul123.github.io/RepulsorPong/Hosting/v0.9/)!  

Controls:
Player 1:
* Left: A
* Right: S
* Speed Up: Left Shift
* Repel: Space

Player 2:
* Left: Left Arrow
* Right: Right Arrow
* Speed Up: Enter
* Repel: Right Shift
